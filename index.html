Digital Clone: Your AI Persona

A sophisticated AI chatbot designed to simulate a specific personality using biographical data, semantic memory, and an emotional matrix.

Powerful Capabilities

Beyond simple chat, Digital Clone offers advanced cognitive features.

🧠

Semantic Memory RAG

Utilizes Retrieval-Augmented Generation with composite scoring (Similarity + Recency) to provide contextually relevant responses based on past interactions.

📂

Document & Image Analysis

Built-in support for analyzing PDFs, text files, and images. The AI can "see" and "read" your files to provide detailed insights and data extraction.

🎭

Personality Matrix

Leverages a complex emotional matrix and biographical indexing to maintain a consistent persona, reflecting specific traits, history, and voice.

Quick Configuration

Get your Digital Clone up and running in minutes by following these simple steps.

  • 1

    API Key Setup

    Enter your OpenAI API key to enable the chatbot's intelligence. You can do this in the chatbot settings or on the Activation Page.

  • 2

    PHP Environment

    Ensure your server supports PHP for file uploads (vupload.php) and memory synchronization (saveMemory.php, getMemory.php).

  • 3

    Biographical Indexing

    Process the bio files to create the semantic embeddings necessary for the persona's memory. Go to Activation
