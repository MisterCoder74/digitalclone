Licensed with attribution.

## Overview
Digital Clone is an AI Persona Chatbot designed to simulate a specific personality using biographical data and an emotional matrix.

## Features
- Single Persona architecture
- RAG with composite scoring (Similarity + Recency)
- Server and local memory synchronization
- Document and Image analyzer
- Web search integration
